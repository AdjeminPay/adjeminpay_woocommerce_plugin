[AdjeminPay](https://adjeminpay.com) vous permet de recevoir des paiements mobile money dans votre site WooCommerce.

== Description ==

AdjeminPay est un service proposé par [Adjemin](https://adjeminpay.com) aux startup, entreprises et institutions financières afin de leur faciliter
 la collecte de fonds sur sites internet et applications mobiles.

Le Plugin Wordpress AdjeminPay intègre les paiements mobile money et carte bancaire dans votre site WooCommerce.
